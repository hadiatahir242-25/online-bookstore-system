# Online Bookstore System

A Java-based online bookstore application developed using object-oriented programming principles. The project features an interactive JavaFX interface for browsing and managing books, with HTML and CSS used to enhance the user interface and styling.

## Features

- Interactive JavaFX graphical user interface
- Browse and manage books
- Object-oriented application design
- User-friendly interface with CSS styling
- Modular project structure

## Technologies Used

- Java
- JavaFX
- Object-Oriented Programming (OOP)
- HTML
- CSS

## Getting Started

1. Clone the repository.
2. Open the project in your preferred Java IDE.
3. Configure JavaFX if required.
4. Build and run the application.
